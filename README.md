# Smart Timetable Management System

A Flask-based web application developed as a college project to automate timetable generation and management for educational institutions. The system provides dedicated dashboards for administrators, faculty members, and students, enabling efficient timetable generation, schedule management, notifications, feedback handling, and PDF export.

## Features

- Role-based authentication (Admin, Faculty, Student)
- Automatic timetable generation
- Faculty schedule management
- Student timetable viewing
- PDF timetable export
- Notification system
- Feedback and issue reporting
- SQLite database integration

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- ReportLab

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/Smart-Timetable-Management-System.git
```

Navigate to the project:

```bash
cd Smart-Timetable-Management-System
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:
http://127.0.0.1:5000
